kozip

키워드로 한국 우편번호를 검색하세요.
도로명 · 지번 주소, 한/영, JSON/CSV 출력 지원.

시작하기 GitHub

설치

# Homebrew (macOS / Linux)
brew install kyungw00k/cli/kozip
# Go 1.21 이상 필요
go install github.com/kyungw00k/kozip/cmd/kozip@latest
# OS/아키텍처 자동 감지 — macOS · Linux · amd64 · arm64
curl -fsSL https://kyungw00k.dev/kozip/install.sh | sh

설치 경로를 변경하려면: INSTALL_DIR=/usr/local/bin curl -fsSL … | sh

빠른 시작

kozip 강남역 키워드로 우편번호 + 도로명 주소 검색
kozip gangnam --lang en 영문 주소로 검색 및 출력
kozip 역삼동 --jibun 지번 주소 표시
kozip 강남역 --lang all 한국어 + 영문 주소 모두 출력
kozip 강남역 -o json JSON 형식으로 출력
kozip 강남역 -o csv > results.csv CSV로 내보내기

출력 예시

$ kozip 강남역

우편번호   주소
──────────────────────────────────────────────────
06164      서울특별시 강남구 강남대로 지하 396 (역삼동)
06232      서울특별시 강남구 테헤란로 지하 211 (역삼동)

기능

🌐

다국어 주소

한국어 · 영문 도로명 / 지번 주소를 모두 지원합니다. --lang ko|en|all

⚡

SQLite 캐시

검색 결과를 로컬 SQLite DB에 캐싱해 반복 조회 속도를 극대화합니다.

📄

멀티포맷 출력

Table · JSON · JSONL · CSV — 파이프라인에 맞는 형식을 선택하세요.

🤖

AI Agent 지원

tool-schema 서브커맨드로 JSON Schema를 제공해 LLM 툴콜에 바로 연동됩니다.

📦

Go 라이브러리

go get 한 줄로 자신의 Go 프로젝트에 우편번호 검색을 내장하세요.

🗺️

지도 링크

카카오맵 · 네이버지도 URL을 결과에 포함해 바로 확인할 수 있습니다.

라이브러리로 사용하기

kozip을 Go 모듈로 임포트해 프로그램에서 직접 우편번호를 검색할 수 있습니다.

패키지 추가

go get github.com/kyungw00k/kozip

예시 코드

import (
    "context"
    "fmt"

    "github.com/kyungw00k/kozip"
)

func main() {
    ctx := context.Background()

    results, err := kozip.Search(ctx, "강남역")
    if err != nil {
        panic(err)
    }

    for _, r := range results {
        fmt.Printf("%s  %s\n", r.Postcode5, r.KoAddress)
    }
}
