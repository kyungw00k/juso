juso

한국 우편번호 검색 CLI & Go 라이브러리

설치

brew install kyungw00k/cli/juso
go install github.com/kyungw00k/juso/cmd/juso@latest
curl -fsSL https://kyungw00k.dev/juso/install.sh | sh

도로명/지번 주소 검색

키워드로 도로명과 지번 주소를 모두 검색합니다. --jibun 옵션으로 지번 주소를 표시합니다.

한/영 다국어 주소

한국어와 영문 주소를 모두 지원합니다. --lang ko|en|all 옵션으로 언어를 선택합니다.

SQLite 캐시 (24시간 TTL)

검색 결과를 로컬 SQLite DB에 24시간 캐싱해 반복 조회 속도를 극대화합니다.

멀티포맷 출력 (table/json/csv)

Table · JSON · CSV 형식을 지원합니다. -o json 또는 -o csv 옵션을 사용합니다.

AI Agent 연동 (tool-schema)

tool-schema 서브커맨드로 JSON Schema를 제공해 LLM 툴콜에 바로 연동됩니다.

셀프 업데이트

juso update 명령으로 최신 버전으로 자동 업데이트할 수 있습니다.

빠른 시작

juso 테헤란로
juso 판교역로 --lang en
juso 테헤란로 --jibun
juso 강남대로 --lang all
juso 테헤란로 -o json
juso 테헤란로 -o csv > results.csv

라이브러리로 사용하기

import "github.com/kyungw00k/juso"

results, err := juso.Search(ctx, "테헤란로")
for _, r := range results {
    fmt.Println(r.Postcode5, r.KoAddress)
}

AI 스킬

juso tool-schema
juso tool-schema search

tool-schema 서브커맨드로 LLM/MCP 도구 호출용 JSON Schema를 내보냅니다. Claude Code 및 AI Agent와 바로 연동할 수 있습니다.
